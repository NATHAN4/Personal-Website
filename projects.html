WatWhenWhere
> Source on Github
> Project Website
I am currently working with a team of four on a web app that allows University of Waterloo students to find common availabilities in their packed schedules. We are using HTML / CSS for the front end, PHP and JavaScript for the backend, and MySQL for the databases. The app also utilizes a Facebook/Google login to allow users easy and quick access to our service, removing the hassle of signing up. My role in the team has been ensuring the login system and authentication works on all pages, and writing all website backend using PHP and JavaScript. The app was launched to the public on January 4'th, 2016 and is in continued development.

Android Note Manager
> Source on Github
Over the summer of 2015, I started and finished development for my Android Note Manager. The Note Manager allows users to upload their school or personal notes to their cloud account and access them wherever they are, whenever they want. All notes are associated with a date, time, title, description and tags. This allows users to sort by date or relevance to a search term. The app is written in Java and XML, with web backend done in PHP and MySQL. User passwords are securely encrypted with a SHA-512 salted hash setup.

LaTeX Notes
> Source on Github
> Project Website
After beginning my first term at the University of Waterloo on a very rough note, I decided to begin typesetting my notes with LaTeX, something I had been meaning to learn how to do for a long time. Although typesetting for 7 lectures a week (only math classes) is a huge time-sink, typesetting makes me re-read my notes as I type which helps a lot with memorization. I also try to explain the more difficult concepts in my own words when I'm typesetting which helps with my own comprehension, and gives me great notes to study from later on. As an incentive to typeset and maintain quality, I've put my notes up on my website for free and accumulated a somewhat-large audience. Over the first term, 1.3GB of notes were downloaded from my site.
Java Grader for Teacher
In high-school, I created an automatic program-marker for my computer science teacher. The program worked by asking for all .java files from all students, Scanner input and expected output. It then compiled the .java files, inputted the Scanner input, checked the output against the expected output, and assigned each student a grade. The application was written in Java.
